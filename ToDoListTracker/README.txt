Instructions to Compile and Run:

1. Open a terminal (Command Prompt, Powershell, etc.).
2. Navigate to the folder where the files are saved.
3. Compile the program:
   javac Task.java TaskManager.java Main.java
4. Run the program:
   java Main

Features:
- Add tasks with a name, priority, and due date.
- View all tasks.
- Mark a task as complete.
- Delete a task.
- Exit the program.

Enjoy using your simple To-Do List Tracker!
